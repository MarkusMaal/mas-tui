## Markuse arvuti juhtpaneel (MasCpanel)

Projekt on hilises Alpha arendusjГӨrgus.

### Kompileerimine

Projekti ehitamiseks on vajalik .NET 10.0 raamistik, SDK ja PowerShell. Kui olete PowerShellis, saate rakenduse kompileerida jГӨrgmiste kГӨsklustega:

```
Set-ExecutionPolicy Bypass -Scope Process
.\build.ps1
```
